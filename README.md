### waste-recognition
一款集垃圾识别，整理地方规范，和小游戏为一体的微信小程序，识别下方二维码即可体验
<img src="https://s1.ax1x.com/2020/04/30/JbKwzd.png" alt="微信小程序二维码" border="0">

#### 1、垃圾识别
根据用户上传的图片，在服务器识别出该图片中的垃圾种类，并返回给用户
#### 2、地方规范整理
根据用户的地理位置，返回所在省份的垃圾分类规范
#### 3、小游戏
贪吃蛇，引导用户去熟悉垃圾的分类。
#### 4、关于我们
本项目已参加互联网+创新大赛。
